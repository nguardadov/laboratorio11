Instalación de PHP y MYSQL en Ubuntu

  1. Abrimos un terminal.
  2. En la terminal actualizamos las dependencias por medio del seguinete comando: sudo apt-get update .

  3. Procedermso a instalar el motor de base de datos de MYSQL, por medio del siguiente comando: sudo apt-get install mariadb-server mariadb-client

    Ahora configuraremos el acceso por contraseña a nuestro servidor MYSQL, ejecutando en la terminal el siguiente comando sudo mysql_secure_installation; acá nos pedira la nueva contraseña para el usuario root.
    Nota: Para las demas opciones las dejaremos por defecto.
    para acceder a nuestro motor de bases de datos abriremos un terminal y ejecutaremos el siguiente comando mysq -u root -p

  4. Instalaremos el servidor de Apache2 para poder trabajar con PHP, para hacerlos debemos ejecutar el siguiente comando: sudo apt-get install apache2

    Para comprobar que se instalo corectamente, debemos ingresar al navegador de nuestra preferencia y en la barra de direcciones, escribimos localhost, nos debera mostrar la pantalla de bienvenida de apache.

  5. Instalaremos php en su ultima version por medio de la siguiente instrucción sudo apt-get install php libapache2-mod-php libmcrypt-dev php-mysql

  6. Para poder instalar phpmyadmin lo hacemos de la siguiente fomra:

    1. Ejecutamos el siguiente comando sudo apt-get install phpmyadmin
